PASOS RAPIDOS (2-3 min)

1) AZURE (OneDrive)
- Abre: Azure Portal > App registrations > New registration
- Name: NEXO-SOBERANO-OneDrive
- Supported account types: Accounts in this organizational directory only
- Redirect URI: Public client/native (http://localhost)
- Crear
- Copia:
  - Application (client) ID  -> client_id
  - Directory (tenant) ID     -> tenant_id
- En Certificates & secrets > New client secret
  - Copia Value -> client_secret
- En API permissions agrega Microsoft Graph:
  - Delegated: User.Read, Files.Read
- Grant admin consent (si aplica)

2) GUARDAR EN ARCHIVO
- Edita: credenciales_microsoft.json
{
  "client_id": "<CLIENT_ID_REAL>",
  "client_secret": "<CLIENT_SECRET_REAL>",
  "tenant_id": "<TENANT_ID_REAL>"
}

- API Library: habilita Google Photos Library API
- OAuth Consent Screen:
  - agrega tu cuenta como Test User
- Credenciales OAuth deben estar en backend/auth/credenciales_google.json
