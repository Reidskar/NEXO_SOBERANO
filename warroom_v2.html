📂 Archivos Recientes (Google Drive)

Conectando a Google Drive...

Paso 1: Descargar Credenciales OAuth
1. Ve a Google Cloud Console (console.cloud.google.com)
2. Crea un proyecto nuevo
3. Habilita Google Drive API
4. En la sección OAuth consent screen:
  • Selecciona External como tipo de usuario.
  • Agrega tu cuenta de Google como test user.
Esto permite autorizar la app aun cuando no ha sido verificada por Google
5. Descarga el JSON en tu máquina
Paso 2: Copiar JSON a la Carpeta Correcta
Copia el archivo JSON descargado (ej: client_secret_xyz.json) a:
backend/auth/credenciales_google.json
Paso 3: Autorizar Nexo Soberano
En terminal, desde la carpeta del proyecto:
python -c "from services.connectors.google_connector import authorize_drive; authorize_drive()"
Paso 4: Verificar Setup
Si la autorización fue correcta, deberías ver archivos en la pestaña "Google Drive".
⚠️ Seguridad: Nunca compartas el archivo de credenciales. Están en .gitignore por defecto.
