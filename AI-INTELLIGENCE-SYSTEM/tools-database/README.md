# Tools Database

Base para registrar herramientas detectadas automáticamente en videos.

## Formato sugerido por entrada

- Nombre de herramienta
- Categoría (AI/API/Automation/Analytics/Infra)
- Casos de uso detectados
- Endpoint o docs oficiales
- Riesgos / permisos requeridos
- Integración sugerida (Discord, n8n, Drive, backend)

## Archivo recomendado

Crear un archivo por herramienta:

- `openai.md`
- `n8n.md`
- `google_drive_api.md`
- `discord_webhook.md`
